# Basic SDK Usage

Demonstrates the core @senzing/sdk workflow: adding records, searching, entity resolution, export iteration, and error handling.

## Prerequisites

- Senzing runtime installed
- `DYLD_LIBRARY_PATH` (macOS) or `LD_LIBRARY_PATH` (Linux) set to the Senzing lib directory
- A pre-initialized Senzing SQLite database at `/tmp/senzing-example.db`

## Run

```bash
npm install
npm start
```
