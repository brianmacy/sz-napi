# Config Management

Demonstrates the Senzing configuration management workflow: creating a config template, modifying it with @senzing/configtool, registering and activating it, and reinitializing the environment.

## Prerequisites

- Senzing runtime installed
- `DYLD_LIBRARY_PATH` (macOS) or `LD_LIBRARY_PATH` (Linux) set to the Senzing lib directory

## Run

```bash
npm install
npm start
```
