## Documentation System

### How documentation is built

Documentation is generated by TypeDoc from type definitions and guide markdown files:

```bash
npx typedoc
```

Output goes to `docs/site/` (gitignored). The `docs.yml` workflow deploys this to GitHub Pages on push to main.

### Documentation structure

```
docs/
  index.md            # Landing page (TypeDoc readme option)
  guides/
    getting-started.md
    config-management.md
    error-handling.md
    deployment.md
    code-snippets.md
  site/               # Generated output (gitignored)
```

### TypeDoc configuration

`typedoc.json` configures:
- **entryPoints**: `packages/sdk/sdk.d.ts`, `packages/configtool/index.d.ts`, `packages/configtool/configtool-docs.d.ts`
- **projectDocuments**: All 5 guide markdown files — these appear in the sidebar alongside the API reference
- **readme**: `docs/index.md` — the landing page

### @example tags

API methods have `@example` code blocks and `@see` links to code snippets:
- **sdk.d.ts** (hand-maintained): ~30 methods across SzEnvironment, SzEngine, SzConfigManager, SzProduct, SzDiagnostic — augmented via `declare module './index'`
- **errors.d.ts** (hand-maintained): Error classes with usage examples
- **configtool-docs.d.ts** (hand-maintained): Top 8 configtool functions with examples

### Guide docs frontmatter

Each guide in `docs/guides/` has YAML frontmatter for TypeDoc:
```yaml
---
title: Getting Started
group: Guides
category: Tutorials
---
```

### Adding @example to an API method

Add to the `declare module './index'` block in `packages/sdk/sdk.d.ts`:
```typescript
interface SzEngine {
  /**
   * Description of the method.
   *
   * @example Example title
   * ```typescript
   * // example code
   * ```
   *
   * @see {@link https://github.com/brianmacy/sz-napi/tree/main/code-snippets/... | snippet name}
   */
  methodName(args): returnType;
}
```

### Known TypeDoc warnings

5 warnings about `documents/*.html` links in `docs/index.md` are expected — TypeDoc validates links before generating the referenced files. The links work correctly in the output.
